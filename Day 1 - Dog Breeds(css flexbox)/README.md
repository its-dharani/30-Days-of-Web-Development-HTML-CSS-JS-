# Day 1 – Flexbox Practice (Card UI)

Today I worked on building a simple card layout using HTML and CSS.

I kept forgetting how flexbox works, especially the main axis and cross axis, so I decided to just play around with it instead of watching more tutorials.

## What I did
- Built a basic card UI  
- Used flexbox for layout  
- Practiced `flex-direction`, `justify-content`, and `align-items`  
- Tried spacing using `gap` and `padding`  

## What I learned
- Flexbox makes more sense when I experiment instead of memorizing  
- `justify-content` controls the main axis  
- `align-items` controls the cross axis  
- Small things like padding and margin make a big difference in UI  

## Thoughts
Still not perfect, but I understand flexbox better than before.
